> Function to remove indentation in multi-lines template literals (string) based on the shortest indented line.

## Installation

```console
$ npm install multili
```

## Usage

```js
import multili from "multili"

multili(
  `
      This is a
        multi-lines
          string
  `
)

// Will produce
/*
This is a
  multi-lines
    string
*/

multili(`
  This is a
  multi-lines
    string
`)

// Will produce
/*
This is a
multi-lines
  string
*/


multili(`
      This is a
  multi-lines
    string
`)

// Will produce
/*
    This is a
multi-lines
  string
*/
```
